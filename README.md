# 🎣 蜘洲钓鱼攻略

珠海桂山岛 · 大蜘洲 · 圆排筏钓黑鲷（黑牛）资料包

## 这个仓库适合谁

适合准备去 **桂山岛大蜘洲圆排** 筏钓、目标锁定 **黑鲷 / 黑牛 / 5斤级大鱼** 的钓友。

## 一页结论

- **最佳月份**：4月明显优于3月，清明前后到4月中旬是主窗口
- **最佳潮汐**：小潮优先，中潮可钓，大潮尽量不去
- **风力红线**：海面 **4级以上不出钓**
- **核心钓位**：两岛之间水道附近、靠岛侧、背流面圆排
- **核心线组**：**全飘 / 无铅优先**，必要时再补 **2-10g 轻坠**
- **核心饵料**：整颗带壳生蚝，重点是**自然下层**
- **核心读口**：快碎乱浅是杂鱼，慢沉稳深才是黑鲷实口

## 使用建议

- **临出发前**：先看主攻略开头的“30秒决策版”
- **判断当天该全飘还是补轻坠**：重点看风力、潮汐、走水和饵下层状态
- **到圆排后认不准是什么鱼在闹**：先翻 `蜘洲常见鱼种与作钓特性详解.md`
- **到圆排后没口或全是碎口**：再看避杂鱼报告里的 7 大策略
- **拿不准 4 月是否值得冲**：看科学论证篇建立预期

## 基本信息

- **位置**：万山群岛 · 大蜘洲附近圆排区
- **交通**：桂山岛码头快艇约 20 分钟
- **钓法**：HDPE 圆排筏钓
- **目标鱼**：黑鲷（黑牛）

祝你这趟不只是“有口”，而是能真正守到大黑牛。
